# OTP Guesser

Very fast tool to bruteforce [One-time pad](https://en.wikipedia.org/wiki/One-time_pad) ciphertext.

## Using

### Find possible variations for `fbjj`

```shell
$ otp-guesser fbjj wordlist.txt
```
